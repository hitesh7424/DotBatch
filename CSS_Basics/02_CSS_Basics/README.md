css gradients types:
1. linear gradients (goes down up left right diagonally)
2. radial gradients ( defined by their center)
3. conic gradients (rotated around a center point)


css Shadow effects:
1. text-shadow
2. box-shadow

text shadow:
- applies to text
- you can add horizontal/vertical shadows
- color can be added
- multiple shadows on 1 text can be added
- question: how can we add border using shadows?

box-shadow:
- applies one or more shadows to an element
- default color of the shadow is text color
- color of the shadow can be chaged
- blur can be added
- spread-radius can be changed
- multiple shadows can be added using comma
